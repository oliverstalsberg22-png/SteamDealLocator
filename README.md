# SteamDiscountFinder

C++ console app that pulls data from the CheapShark API using CPR.
Features: 

## Features
- Browse current Steam game deals
- Filter deals by price, deal rating, and percent off
- View sale price vs normal price

## Dependencies
- C++17 or later
- [CPR](https://github.com/libcpr/cpr)
- [vcpkg](https://github.com/microsoft/vcpkg)
- [CheapShark API](https://apidocs.cheapshark.com/)

## Setup
1. Clone the repository
   git clone https://github.com/yourusername/SteamDiscountFinder.git
2. Install dependencies via vcpkg
   vcpkg install cpr
3. Open SteamDiscountFinder.slnx in Visual Studio
4. Build and run
